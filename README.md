## Quick Start

```bash
# clone our repo
$ git clone https://github.com/Big-Silver/Angular-Drag-and-Drop.git angular-Drag-and-Drop

# change directory to your app
$ cd angular-Drag-and-Drop

# install the dependencies with npm
$ npm install

# start the server
$ npm start
```
